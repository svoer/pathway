Application interne Enovacom

Enovacom Pathway est une application web interne développée pour un usage exclusif au sein d'Enovacom. Cette application n'est pas accessible au public.

1. Éditeur de l'application

Société : Enovacom

Application : Enovacom Pathway

Type : Application web interne (Flask)

Usage : Réservé aux collaborateurs Enovacom

Contact technique : sebastien.voerman@enovacom.com

2. Hébergement

Type d'hébergement : Infrastructure interne Enovacom

Serveur : Serveur Flask local/interne

L'application est hébergée sur l'infrastructure informatique d'Enovacom et n'est accessible que depuis le réseau interne de l'entreprise.

4. Protection des données personnelles

Conformément au Règlement Général sur la Protection des Données (RGPD) et à la loi Informatique et Libertés, cette application respecte la confidentialité des données des collaborateurs Enovacom.

Données collectées et stockées

Enovacom Pathway collecte et stocke les données suivantes :

  • Clé API Mistral AI : Stockée localement dans le navigateur de l'utilisateur (localStorage)
  • Historique des prompts : Les 10 derniers prompts stockés localement dans le navigateur
  • Préférences utilisateur : Thème, police, couleurs - stockées localement dans le navigateur

Important : Toutes les données sont stockées localement dans votre navigateur (localStorage). Aucune donnée n'est stockée sur un serveur central. Les données restent sur votre poste de travail.

Utilisation de l'API Mistral AI

Lorsque vous utilisez la fonctionnalité de génération par IA :

  • Vos prompts sont envoyés directement à l'API Mistral AI pour traitement
  • Aucune donnée n'est interceptée ou stockée par Enovacom lors de ce transfert
  • Vous utilisez votre propre clé API Mistral AI
  • Consultez la politique de confidentialité de Mistral AI : https://mistral.ai/terms/

Responsable du traitement

Enovacom est responsable du traitement des données dans le cadre de cette application interne.

Exercice de vos droits

En tant que collaborateur Enovacom, pour toute question relative à vos données personnelles, contactez : sebastien.voerman@enovacom.com

5. Cookies

Ce site utilise le stockage local du navigateur (localStorage) pour sauvegarder vos préférences et paramètres. Aucun cookie de suivi ou de publicité n'est utilisé.

Données stockées localement

  • Paramètres API Mistral (base_url, api_key)
  • Historique des prompts (10 derniers)
  • Préférences de personnalisation (thème, couleurs, police)

Vous pouvez supprimer ces données à tout moment en effaçant le localStorage de votre navigateur ou en utilisant le bouton "Effacer" dans l'historique.
